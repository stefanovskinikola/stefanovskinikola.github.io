# Nikola Stefanovski - Web Developer Portfolio

A minimalistic, interactive personal portfolio built to showcase my web development projects, technical skills, and experience. This project was developed as a homework assignment for the Web Development Academy, with a strong focus on clean architecture, robust User Experience (UX), and custom Vanilla JavaScript logic, all wrapped in a fully responsive design.

- **Minimalistic & Responsive Design:** A clean, distraction-free UI that adapts flawlessly to any screen size, ensuring a premium viewing experience on both mobile and desktop.

- **Modular Architecture:** A frontend built with modern CSS variables (custom properties) and a JavaScript codebase cleanly organized into ES6 modules for maintainability and scale.
